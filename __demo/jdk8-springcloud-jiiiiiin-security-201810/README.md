# jiiiiiin-security

一个前后端分离的内管基础项目

# 计划
| 功能 | 完成状态 | 简介 |
| ------ | ------ | ------ |
| 操作员管理 | 2% | 权限管理的操作员管理模块 |
| 角色管理 | 2% | 权限管理的角色管理 |
| 资源管理 | 30% | 权限管理的资源管理模块 |
